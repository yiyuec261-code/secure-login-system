# Secure Login System

A simple secure login system built with HTML, CSS, and JavaScript.

## Features

- User registration interface
- Password strength detection
- Responsive cyber-style UI
- Real-time password feedback

## Technologies Used

- HTML
- CSS
- JavaScript

## Project Goal

This project was created to explore the relationship between cybersecurity, user interaction, and frontend development.

## Future Improvements

- Add Flask backend
- Connect SQLite database
- Implement password hashing
- Prevent SQL injection attacks
- Add authentication system
